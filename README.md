## 📌 Sobre o Projeto

Este projeto foi desenvolvido como parte da disciplina **Projeto Integrador III-B**, do curso de **Análise e Desenvolvimento de Sistemas**, com foco na aplicação prática dos conhecimentos adquiridos ao longo da graduação.

A proposta consiste na criação de um sistema web para gerenciamento de tarefas, desenvolvido no contexto de uma **atividade extensionista**, em parceria com uma microempreendedora do ramo de artesanato, especializada na produção de adesivos e pinturas para unhas.

O sistema tem como objetivo principal auxiliar na organização das atividades, controle de prazos e melhoria da produtividade da empresa parceira.

---

## 🎯 Objetivos

### 🎯 Objetivo Geral

Desenvolver uma aplicação web para gerenciamento de tarefas, proporcionando organização, controle e acompanhamento das atividades.

### 📍 Objetivos Específicos

- Organizar tarefas de forma eficiente  
- Permitir o acompanhamento das atividades  
- Controlar prazos de entrega  
- Melhorar a produtividade  
- Aplicar conceitos de desenvolvimento web  
- Utilizar práticas de gerenciamento de projetos

## ⚙️ Funcionalidades

O sistema possui as seguintes funcionalidades:

- 👤 Cadastro de usuários  
- 🔐 Autenticação de login  
- 📁 Criação de projetos  
- 📝 Criação de tarefas  
- ✏️ Edição e exclusão de tarefas  
- 📊 Acompanhamento do andamento das tarefas  
- 📌 Organização em quadro Kanban:
  - A fazer  
  - Em andamento  
  - Concluído  
- ⏰ Controle de prazos  
- 💾 Persistência de dados com localStorage  

---

## 📊 Metodologia de Desenvolvimento

O projeto foi desenvolvido com base em metodologias ágeis, permitindo maior flexibilidade e organização durante o processo.

A ferramenta **Trello** foi utilizada para gerenciamento das tarefas, organizadas no modelo Kanban:

- 📌 A fazer  
- 🔄 Em andamento  
- ✅ Concluído  

Essa abordagem facilitou o acompanhamento do progresso e a organização das atividades.
